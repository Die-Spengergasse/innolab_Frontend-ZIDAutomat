Wie borge ich einen Artikel aus?

Gehen Sie auf die "/home" Page(Navbar ZID Automat). Dort sehen Sie alle verfügbaren Artikel. Mit drücken auf den Artikel kommen Sie zu einem Dialog. Dort gibt es eine Sektion namens Ausborgen. Über den Datepicker können Sie auswählen, wann Sie den Artikel zurückgeben wollen. Nun müssen Sie noch das Einverständnis geben, dass Sie sich bewusst sind, dass Sie gerade einen Artikel ausborgen, via des Toggle Buttons. Mit drücken auf den “Ausborgen” Button wird das Ausborgen gestartet. Sie erhalten einen QR- Code. Mit diesem QR- Code können sie zum Automat gehen und ihn einscannen. Der Artikel sollte darauf hin ausgegeben werden

Wo gebe ich den ausgeborgten Artikel zurück?

ausgezeichnete Frage!!!!

Wo sehe ich meine aktuellen Ausleihen?

Ihre Artikel, die noch nicht abgeholt worden sind, können sie auf der ActiveQrCodePage ansehen( Navbar Qr-Code Icon oder “/activeQrCodes”) Dort haben Sie die möglichkeit sich Informationen über die Ausleihe anzuschauen und den QR-Code noch einmal anzeigen zu lassen

Wie lange habe ich Zeit, um einen Artikel abzuholen?

Generell gilt, dass nach dem Drücken des 'Ausborgen'-Buttons der Artikel für eine Stunde reserviert ist. Nach dieser Zeitspanne vergeht die Reservierung und man müsste den Reservierungsprozess von neuem starten.

Was passiert, wenn ich den Artikel verliere/ nicht zurück gebe?

Wenn ein Artikel abhandengekommt oder beschädigt wird, muss dieser Schaden kompensiert werden. Der Betrag, der bei einer beschädigung zu bezahlen ist, findet sich neben jedem Produkt

Kann ich mir eine Historie meiner Ausleihen ansehen?

Ja auf der “history”-page(zweiter Button von links in der Navbar)

Ab wann fallen Kosten an?

Sollten Sie das Produkt beschädigt, oder gar nicht zurück geben, fallen Kosten an.
